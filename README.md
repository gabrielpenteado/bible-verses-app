### Installation

1. Clone the repository<br>
   `git clone https://github.com/gabrielpenteado/bible-verses-app`

2. Access the project folder in the terminal

3. Install all packages dependencies<br>
   `npm install`

4. Start using<br>
   `npm run dev`
